                                  screenRecorder

                                  $h. screenRecorder

                                  Version:
                                  • 1.0.0
                                  Author:
                                  • Inswave Systems

                                  디바이스(폰,테블릿)에 화면 녹화/녹음에 필요한 녹화,저장,취소,일시정지 등 API를 제공한다.

                                  Methods

                                  (static) deleteAllFile(callback, options)

                                  녹화된 파일 및 폴더를 전부 삭제한다.

                                  Example
                                  var callback = function(result){ console.log(result); };
var options = {'isInternal':'true'};
$h.screenRecorder.deleteAllFile(callback,options);

                                  Name Type Description
                                  callback function
                                  options Object

                                  object

                                  • isInternal 삭제할 파일 저장소 중 내부/외부 저장소 (Default: true)

                                  (static) recordScreen(callback, options)

                                  option의 설정에 따라 화면 녹화를 시작한다.

                                  Example
                                  var callback = function(result){ console.log(result); };
var options = {'fileName':'screenRecorderExample', 'isInternal':'true', 'audio':'false', 'saveToGallery':'true'};
$h.screenRecorder.recordScreen(callback,options);
                                  Parameters:
                                  Name Type Description
                                  callback function
                                  options Object

                                  object

                                  • fileName 파일 이름
                                  • audio 오디오 녹음 여부(true or false) Default: false
                                  • isInternal 녹음 파일을 내부/외부 저장소중 저장할 곳을 설정 (true or false) Default: true / 내부 저장소
                                  • saveToGallery 녹음 파일 앨범에 저장유무 (true,false)
