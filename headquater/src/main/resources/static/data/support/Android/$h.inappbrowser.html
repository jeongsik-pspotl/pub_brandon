                                  inappbrowser

                                  $h. inappbrowser

                                  Version:
                                  • 1.0.0
                                  Author:
                                  • Inswave Systems

                                  webpage를 별도에 webview로 호출하는 API를 제공한다.

                                  Methods

                                  (static) open(URL, type, option)

                                  inappbrowser 플러그인을 통해서 별도 웹뷰를 붙여서 화면을 띄우거나 모바일브라우저를 호출한다.

                                  Example
                                  $h.inappbrowser.open("https://www.inswave.com","_system");
                                  Parameters:
                                  Name Type Description
                                  URL String

                                  화면URL
                                  type String

                                  "_blank": 웹뷰 , "_system": 모바일브라우저
                                  option String

                                  옵션 예) "locoation=yes"
