speech

$h. speech

Source:
  • wmatrix.js, line 702
Version:
  • 1.0.0
Author:
  • Inswave Systems

앱에 읽기/말하기 기능을 제공하는 플러그인으로 API를 제공한다. 제공되는 API로는 speakTTS, stopTTS, startSpeechToText, stopSpeechToText 기능들이 있다.

Methods

(static) speakTTS(callback, options)

Source:
  • wmatrix.js, line 713

message 를 음성으로 읽어준다.

Example
var callback = function(result){ console.log(result); };
var opt = {"language": "ko", "message" : "hello, world", "speechSpeed" : 0.9 };
$h.speech.speakTTS(callback,opt);
