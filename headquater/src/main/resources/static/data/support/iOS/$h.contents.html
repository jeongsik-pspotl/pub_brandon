$h. contents

Version:
  • 1.1.2

Content관련 API를 제공하는 플러그인으로 image picker, file picker, fileDownload 기능을 제공한다.

Methods

(static) fileDownload(callback, options)

Source:
  • wmatrix.js, line 1316

url에 파일을 다운로드한다.

Example
var callback = function(result){ console.log(result); };
var options = {"url":"http://localhost:8080/download/test.xlsx","progress":"true","open":"true","notify":"true", "location":"Download", "requestType" : "get", "fileName" : "test.xlsx"}

= 다운로드가 완료되면 해당 파일을 open할지 여부 (true or false)

= 다운로드가 완료되면 toast로 다운로드 완료 여부를 알려줄지 여부 (true or false)
location String

= 다운로드가 될 위치 지정 (default : Download)
requestType String

= 다운로드 request Type (get or post)
fileName String

= 다운로드 후 저장될 파일 이름 (post의 경우 필수)
